## ADDED Requirements

### Requirement: Search Filtering by Breed
The system SHALL allow users to filter search results by one or more breeds.

#### Scenario: Filter by Labrador
- **WHEN** user selects "Labrador" as the breed filter
- **THEN** only Labrador profiles within the search radius and matching intent are returned

### Requirement: Search Filtering by Size
The system SHALL allow users to filter search results by dog size (e.g., small, medium, large).

#### Scenario: Filter by Small dogs
- **WHEN** user selects "Small" as the size filter
- **THEN** only small dog profiles are shown in the search results
